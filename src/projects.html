Better D2L

BetterD2L was a project done by myself and a group of four for CalgaryHacks 2021, as University of Calgary students, all course content is hosted on D2L (Desire to Learn), we wanted to take this platform and incorporate some of our ideas on how we could imporve it. Overall, it was a great learning experience and here are just some key contributions I made to the project...

1) Developed a full-stack web application using Flask and Vue.js with tailwind for the UI.
2) Implemented REST API to connect front end to back end using Flask.
3) Integrated front end with API to allow students to submit assignment directly from our application.
4) Used SQLite3 database to store a student’s course related information such as exam dates and assignment deadlines.
